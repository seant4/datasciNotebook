I use Vim because:

# The following is a reference for basic usage of my vim setup

## Nvim-R
- Plugin i use to use R in vim
- Start r:    \rf
- Close r:    \rq
- Send file:  \aa
- Send block: \bb
- Send Chunk: \cc
- Send func:  \ff
- Send sele:  \ss
- Send para:  \pp
- Send line:  \l

## Telescope
Useful for finding files
- Ctrl - T:  Open
- Ctrl - G:  Git
- Ctrl - S:  Search

## Harpoo
Project navigation
- Ctrl - H: Open harpoon menu
- Ctrl - A: Add file to menu
